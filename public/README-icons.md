# PWA Icons

This directory contains PWA icons for the Chaldean World app.

## Current Icons

- `pwa-icon.svg` - SVG icon (primary, works in most browsers)
- `pwa-192x192.png` - Placeholder PNG (needs to be replaced)
- `pwa-512x512.png` - Placeholder PNG (needs to be replaced)

## Generating Proper PNG Icons

To generate proper PNG icons:

1. Open `scripts/create-pwa-icons.html` in a browser
2. Click "Download Icons" button
3. Replace the placeholder PNG files in the `public/` directory

Alternatively, use an online tool like:
- https://realfavicongenerator.net/
- https://www.pwabuilder.com/imageGenerator

Or use the SVG icon (`pwa-icon.svg`) as a source and convert it to PNG at 192x192 and 512x512 sizes.
